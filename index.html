// Questions
Does it search by name or by content?

By name. Everyfile is built to find any file by its name, instantly — the thing Spotlight is surprisingly bad at. It indexes file names (and paths), not the text inside documents. Content search is on the roadmap for a later version.

Does it index iCloud Drive?

Yes. iCloud Drive files live under a folder most search tools quietly skip, so files you saved on another Mac often go missing. Everyfile indexes them too, so they show up alongside everything else.

How fast is it, really?

Instant. On a real home folder of 809,043 files, the full index builds in about 23 seconds and queries return in a p95 of 0.3 ms — results appear faster than you can finish typing. The speed is the whole point.

Does it update automatically?

Yes. After the first build, a lightweight watcher keeps the index fresh in real time — new, moved, renamed and deleted files are picked up automatically. You never have to re-index by hand, and a file you just saved shows up in your next search.

How do search operators work?

Type them right alongside the file name. Use kind: to filter by type (image, pdf, video, doc, code), date: for recency (today, week, month), size:>10mb for size, and path: to restrict to a folder. Combine them freely — for example budget kind:pdf date:month finds PDFs named "budget" touched in the last month.

Can I preview a file before opening it?

Yes. Select any result and press Space for a QuickLook preview — PDFs, images, video, code and more — exactly like Finder. Glance to confirm it's the right file, then hit return to open it. You can also drag a result straight out of the panel into another app, like attaching it to an email.

Will it slow down my Mac or hog CPU?

No. The index is built once and then kept fresh by a lightweight watcher that only reacts to the files that actually change. When you're not searching, Everyfile sits idle at effectively 0% CPU.

Is it a subscription?

No. One-time purchase, yours forever. You get free updates within the same major version (all of 1.x). Major upgrades may be paid, and only if you choose to take them.

How many Macs can I use it on?

Up to 2 Macs you own, with a single license. Enter the key on each and you're set.

What about my privacy?

Everything runs on your Mac. The index — which includes your file names and paths — is stored private to your account (permissions 0600) and never leaves the device. No account, no sign-up, zero telemetry.

Which macOS do I need?

macOS 14 Sonoma or later, on both Apple Silicon and Intel Macs.

Absolutely — a 14-day money-back guarantee, no questions asked. Payments are handled by Paddle; just reply to your receipt and we'll sort it out.
